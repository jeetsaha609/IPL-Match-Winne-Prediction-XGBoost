# 🏏 IPL Match Winner Prediction using XGBoost

## 📌 Project Overview
This project predicts IPL match winners using machine learning and advanced feature engineering.

Features used:
- Team recent form (weighted)
- Head-to-head stats
- Venue advantage
- Toss win percentage
- Player impact (batsman & bowler)

---

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib

---

## 🤖 Model
- XGBoost Classifier
- Time-based split (no data leakage)

---

## 📊 Key Features
- Toss winner
- Venue advantage
- Team strength
- Player impact

---

## 📈 Results
Accuracy: 25-30% accuracy using XGBoost with engineered features like (team form, toss advantage, and player impact.) Model performance can be further improved with hyperparameter tuning and additional data.
